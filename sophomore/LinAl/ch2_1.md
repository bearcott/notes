# Chapter 2.1

*covered how to multiply matrices*

*basic algebraic operations*

*matrix identity*

    [1 0 0]
    [0 1 0]
    [0 0 1]

`Am*n` * `In*n` = `Am*n`
`Im*m` * `Am*n` = `Am*n`

*matrix transposition*
- flip the matrix about the diagonal.
  - `Am*n`^t = An*m

    (A^t)^t = A
    (A + B)^t = A^t + B^t
    (cA)^t = c(A^t)
    (AB)^t = B^t * A^t
